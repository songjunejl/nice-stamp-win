# Nice Stamp v9.0.0 (Windows)

Gesture-based timestamp utility with musical chord audio feedback.

## Features (v9.0.0)

- **Tap**: Insert timestamp at cursor
- **Hold (500ms)**: Quick Select format picker
- **Audio Feedback**: Musical chords from modifier key combinations

## Technology Stack

- .NET 8.0 / WPF
- NAudio for audio synthesis
- Windows low-level keyboard hooks

## Build

```powershell
dotnet build
dotnet run --project NiceStamp
```
